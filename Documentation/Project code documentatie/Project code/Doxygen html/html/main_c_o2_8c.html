void HAL_TIM_PeriodElapsedCallback ( TIM_HandleTypeDef * htim)

Callback-functie voor periode-afgelopen gebeurtenissen van de timer.

Deze functie wordt aangeroepen wanneer de periode van de timer is verstreken. Het veranderd de frequentie en hardheid van de buzzer naar aanleiding van de status hiervan.

Inleiding

Op deze pagina is de volledige implementatie te vinden van de Wemos sockets, die verbonden zijn aan de Wemos/beheerders-interface Raspberry Pi.
Voor de verbinding met de Wemos/beheerders-interface Raspberry Pi wordt er gebruik gemaakt van de WemosClient en WemosServer subklasses. Deze klasses zijn afgeleid van een WiFiClient en WiFiServer klasse uit een library.

Klasses

Om de Algemene Wemos main code correct te laten werken zijn er twee klasses geschreven:

  • WemosClient
  • WemosServer

De klasses includen

Om de klasses te kunnen gebruiken, moeten de volgende stappen gevolgd worden:
1) Ga naar de plek waar de Arduino IDE bestanden opgeslagen worden.
2) Ga naar de libraries submap.
3) Maak een nieuwe map voor de klasses (bijvoorbeeld Wemos klasses).
4) Plaats de .h en .cpp bestanden van deze klasses in deze map.
Nu kunnen de .h bestanden ge-include worden, om de klasses te kunnen gebruiken in de Arduino code.

Main code

Om de documentatie van deze main te zien, zie: Algemene Wemos main.

void setup() {
Serial.begin(115200);
WiFi.begin(ssid, password);
while(WiFi.status() != WL_CONNECTED) {
Serial.println("Connecting to the WiFi");
delay(1000);
}
Serial.println("Connected to the WiFi");
server.begin();
Serial.println("Server started");
Serial.println("IP: ");
Serial.println(WiFi.localIP());
client.sendIP("Wemos1");
client.sendIP("Wemos2");
client.sendIP("Wemos3");
Serial.println("Waiting for request");
}
void loop() {
serverCode();
if (Serial.available() > 0) {
String inlees = Serial.readStringUntil('\n');
Serial.println("Ontvangen data: " + inlees);
client.sendInput(inlees);
}

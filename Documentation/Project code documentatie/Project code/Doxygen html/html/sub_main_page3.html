Inleiding

Op deze pagina is de volledige implementatie te vinden van de QT beheerders-interface. Deze beheerders-interface geeft een grafische weergave van de huidige staat van het systeem.
Voor de verbinding met de Wemos/beheerders-interface Raspberry Pi wordt er gebruik gemaakt van de QTSocketClient en QTSocketServer klasses.

Het programma builden

Om het programma te runnen, kunnen de volgende stappen gevolgd worden:

  1. Open QT creator.
  2. Open de UI build.
  3. Klik op de play knop links onder.

Klasses

Om de QT main code correct te laten werken, wordt er gebruik gemaakt van een aantal klasses:

  • MainWindow
  • Deur
  • Draaideur
  • Schuifdeur
  • QTSocketClient
  • QTSocketServer

Main code

Om de documentatie van deze main te zien, zie: QT main.
Hieronder is de losse main te zien:

int main(int argc, char *argv[])
{
QApplication a(argc, argv);
MainWindow w;
w.show();
return a.exec();
}

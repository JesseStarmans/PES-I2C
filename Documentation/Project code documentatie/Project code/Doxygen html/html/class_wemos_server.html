String WemosServer::receiveRequest ( )

Checkt of er data gelezen kan worden van de client verbinding.

Deze functie checkt of er data in te lezen is van de client en als dit zo is, leest deze de volledige string in die verstuurd is. Deze string wordt dan weer teruggegeven.
